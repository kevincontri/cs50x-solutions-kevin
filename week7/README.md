# Week 7 - SQL

This week introduced relational databases and SQL, while focusing on how to store, query, and analyze structured data efficiently.
The problem sets emphasized writing precise SQL queries, understanding relationships between tables, usage of SQLite3 syntax, and extracting information from real-world datasets.

-**/songs**: Queries a music database to analyze song characteristics using filtering, aggregation, and sorting in SQL.
-**/movies**: Extracts information from a relational movie database by joining multiple tables and applying conditional queries.
-**/fiftyville**: Solves a mystery by combining data from multiple database tables using joins and subqueries.

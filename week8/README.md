# Week 8 - HTML, CSS, JavaScript

This week introduced front-end web development, focused on building interactive webpages using HTML for structure, CSS for styling, and JavaScript for client-side behavior.
The problem sets emphasized user interaction, dynamic content updates, and integrating logic directly into the browser.

- **trivia/**: Implements an interactive trivia webpage using JavaScript to handle user input, validate answers, and dynamically update the interface.
- **homepage/**: Designs a personal website using HTML and CSS, demonstrating layout, styling, and responsive design principles.

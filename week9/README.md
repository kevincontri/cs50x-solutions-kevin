# Week 9 - Flask

This week introduced web application development using Flask, focused on integrating Python, SQL, and HTML to build dynamic, data-driven web applications.
The problem set emphasized routing, form handling, session management, and connecting a backend server to a relational database.

- **birthdays/**: Implements a web application that stores and displays user birthdays using Flask, SQL, and form submissions.
- **finance/**: Implements a stock trading web application that allows users to register, log in, quote stocks, buy and sell shares, and view transaction history using Flask and SQL.
